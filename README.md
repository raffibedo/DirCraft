# DirCraft

![Version](https://img.shields.io/npm/v/dircraft)
![License](https://img.shields.io/npm/l/dircraft)

DirCraft is a CLI tool designed to transform ASCII tree formatted project structures into real file systems. It's especially powerful for converting project structures generated by AI Large Language Models (LLMs) into actual working directories.

### Basic Commands

```bash
# View help
dircraft --help

# Basic usage (with interactive confirmation)
dircraft structure.txt

# Specify output directory
dircraft -o ./destination-directory structure.txt

# Skip confirmation (useful for scripts)
dircraft -y structure.txt

# Provide structure directly from command line (ideal for AI outputs)
dircraft -s "my-project/
â”œâ”€â”€ src/
â”‚   â””â”€â”€ index.js
â””â”€â”€ package.json"
```

### Available Options

- `-h, --help`: Shows help
- `-y, --yes`: Skips confirmation
- `-o, --output <dir>`: Specifies the output directory (default: current directory)
- `-s, --structure <text>`: Provides the structure directly as text instead of from a file

## Advanced Usage

### Component Generation Script

DirCraft comes with a Next.js 15 component generator example. Run it with:

```bash
node ./node_modules/dircraft/examples/demo.js
```

This interactive script will:

1. Prompt for component details
2. Generate an appropriate directory structure
3. Create all necessary files for a Next.js component

### Integration with AI Workflows

For the most efficient workflow:

1. Ask an AI to generate a project structure for your specific needs
2. Copy the ASCII tree structure directly from the chat
3. Use dircraft with the `-s` flag to create the structure instantly
4. Ask the AI to help you populate key files in the new structure

This workflow eliminates the manual step of creating directories and files, allowing you to focus on the actual code.
